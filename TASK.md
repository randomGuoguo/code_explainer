# TASK

## 0. 目标
- 写一个自动解释python函数的.py
- 希望运行该.py再输入待解析的.py文件名，实现自动在函数体内补全docstring
- 同时新生成一个.md，包含两部分，第一部分说明该.py主要完成什么功能，第二部分展示一个函数概要表格，包含函数名、入参、返回值、功能概述

## 1. 背景
- 公司有很多存量代码，但是缺乏注释，难以理解
- 自己新写的函数越来越多，没有维护，有时会重复造轮子

## 2. 技术路径
1) 读取.py，用 ast 模块提取所有函数、类、注释
2）通过线上大模型接口每次读取.py文件内容，通过prompt完成上述函数docstring补全+概要md生成
